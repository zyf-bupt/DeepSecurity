### 二、仅主机模式下的运行方案：

---

思路：VM 只需要能访问宿主机（Host-Only 一定可以），不需要直达 10.21 网段。

#### 1、配置宿主机：

* 在宿主机（Windows）上做一个端口转发：
	把 `宿主机:15000` 转发到 `<SERVER_IP>:5000`

以管理员 PowerShell 执行：

```bash
netsh interface portproxy add v4tov4 listenaddress=0.0.0.0 listenport=15000 connectaddress=<SERVER_IP> connectport=5000
```

* 放行宿主机防火墙的 15000 端口（入站）
	（Windows Defender 防火墙 -> 入站规则，新建端口 15000 允许）

* 在 VM 里访问宿主机的 Host-Only IP（VMware 一般是 `192.168.86.1` 或 `192.168.86.2`，你可以在宿主机 `ipconfig` 看 VMware Network Adapter VMnet1 的 IP）

#### 2、VM 里测试：

```bash
curl -v http://192.168.86.1:15000/behavior/status
```

* 然后 VM 运行 agent 改成指向宿主机：

```bash
python utils/behavior_monitor/client_agent.py --server http://192.168.86.1:15000 --verbose
```
