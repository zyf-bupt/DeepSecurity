## 本项目的创新点

### 1. 三源数据融合检测
不同于 LLM-APTDS 仅依赖日志，本项目同时接入 Windows Event Log、终端行为监控（进程/文件/网络/DNS）和网络流量（dumpcap实时抓包），通过 ATT&CK 规则引擎和 DeepSeek LLM 实现多源数据的融合分析。

### 2. LLM增强的多智能体协作捕获
AgentStalker 的智能体使用规则引擎，本项目将四个 Agent（证据收集、攻击链构建、验证、裁决）全部接入 DeepSeek LLM，每个 Agent 独立调用大模型进行分析，最终仲裁 Agent 综合研判。

### 3. 端到端一键部署
多数开源项目需要复杂的环境配置，本项目实现 `python app_launcher.py` 一键启动 Flask + Vite 前端，自动检测 SQL Server/Neo4j 可用性并降级到内存模式。

### 4. 真实数据 + 模拟场景双模运行
既支持内置 APT/AI 攻击场景快速演示，也支持真实 Windows Agent 监控 + dumpcap 抓包，实现从演示到生产的平滑过渡。
